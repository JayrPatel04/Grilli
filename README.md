# Grilli - Restaurant Website

A modern and fully responsive restaurant website built using HTML, CSS, and JavaScript.  
Designed to provide a smooth user experience across all devices.

## Live Demo
👉 https://grilli-delta.vercel.app/

## Features
- Responsive design (mobile, tablet, desktop)
- Clean and modern UI
- Smooth scrolling and interactive elements
- Well-structured and reusable code

## Getting Started

### Prerequisites
- Git installed → https://git-scm.com/downloads

### Installation

```bash
git clone https://github.com/JayrPatel04/Grilli.git
cd Grilli
```

Open `index.html` in your browser.
